# 📄 Chat with PDF using Gemini

A smart chatbot that lets users upload PDFs and ask questions based on the content. Powered by **Gemini API**, **LangChain**, **FAISS**, and **Streamlit**.

- 🔗 [Live Demo](https://pdfchatbot-mgpvgpmx6ilcgdsxqerfrw.streamlit.app)
- 📂 [Portfolio](https://sankethhonavar.github.io/Portfolio/)
- ## 🛠 Tech Stack
- **Python**
- **Streamlit**
- **Google Gemini API**
- **LangChain**
- **FAISS (Vector DB)**

## 🔍 Features
- Upload any PDF and ask questions about it.
- Uses **vector embeddings** for chunk retrieval.
- Integrated with **Gemini Pro** for natural language answers.
- Fast & user-friendly UI using Streamlit.
- Designed for simple deployment on **Streamlit Cloud**.
